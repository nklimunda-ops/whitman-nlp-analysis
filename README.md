# whitman-nlp-analysis
Natural Language Processing project analyzing semantic fields, metaphor, and bodily metonymy in Walt Whitman using Python and spaCy.
## Technologies Used

- Python
- spaCy
- Natural Language Processing (NLP)
- Literary Analysis
- Digital Humanities

## Objectives

- Detect semantic fields
- Explore bodily metonymy
- Identify metaphorical patterns
- Combine computational linguistics and literary studies
